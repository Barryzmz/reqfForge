# Glossary

| Term | 定義 | 來源 | 備註 |
| --- | --- | --- | --- |
| 特休 | 法定特別休假，依年資累積天數，員工有權自由安排 | SRC-001 | 年資對照表待 SRC-002 確認 |
| 病假 | 員工因傷病就醫或療養所請假別，法定上限一年 30 天 | SRC-001 | 是否需附診斷書待確認（Q-001） |
| 事假 | 員工因個人事務所請假別，法定上限一年 14 天 | SRC-001 | |
| 喪假 | 員工因直系或旁系親屬死亡所請假別，天數依親等而異 | SRC-001 | 親等對照規則待確認（Q-007） |
| 主管（Manager） | 負責審核所屬員工請假申請的角色 | SRC-001 | 跨部門主管情況待確認（Q-003） |
| HR | 人力資源部門成員，負責查閱全員假期紀錄與匯出報表 | SRC-001 | |
| 員工（Employee） | 系統的主要使用者，負責發起請假申請 | SRC-001 | |
| 代理主管 | 主管不在時，暫時接替主管執行審核的人員 | SRC-001 | 機制尚未確認（Q-004） |
| 跨部門員工 | 同時屬於兩個部門的員工 | SRC-001 | 審核規則尚未確認（Q-003） |
| 催審提醒 | 主管超過指定時間未審核時，系統主動發出的通知 | SRC-001 | 通知方式尚未確認（Q-002） |
