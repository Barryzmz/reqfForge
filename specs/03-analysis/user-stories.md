# User Stories

| Story ID | User Story | 關聯需求 | 優先級 | 狀態 |
| --- | --- | --- | --- | --- |
| US-001 | As an Employee, I want to submit a leave request with dates, type, and reason, so that I can formally apply for leave without using Line. | FR-001 | High | Confirmed |
| US-002 | As an Employee applying for sick leave over 3 days, I want to upload a medical certificate, so that my application meets the required documentation rule. | FR-002、BR-007 | High | Confirmed |
| US-003 | As an Employee, I want to cancel a pending leave request, so that I can withdraw an application I no longer need before it is reviewed. | FR-007、WR-005 | Medium | Confirmed |
| US-004 | As a Manager, I want to approve or reject leave requests from my team, so that I can manage my team's attendance through the system. | FR-003、WR-002、WR-003 | High | Confirmed |
| US-005 | As a Manager, I want to receive a reminder when a leave request has been pending for more than 3 days, so that I do not miss requests that need my action. | FR-004、BR-006 | High | Confirmed |
| US-006 | As a Manager, I want to assign a delegate who can approve leave requests on my behalf during my absence, so that team leave requests are not blocked when I am unavailable. | FR-008、BR-010 | Medium | Confirmed |
| US-007 | As HR, I want to view all employees' leave records, so that I have a complete picture of leave usage across the company. | FR-005 | High | Confirmed |
| US-008 | As HR, I want to export leave records as an Excel report, so that I can analyse leave data outside the system. | FR-006、DR-011 | Medium | Confirmed |
