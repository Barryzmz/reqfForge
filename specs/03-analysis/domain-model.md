# Domain Model

根據已確認需求整理，僅包含需求中有明確依據的實體與屬性。

## Entities

| Entity ID | 實體名稱 | 描述 | 關聯資料需求 |
| --- | --- | --- | --- |
| ENT-001 | Employee（員工） | 系統的基本使用者單位，角色可為 Employee / Manager / HR | DR-009 |
| ENT-002 | Department（部門） | 員工所屬的組織單位，跨部門員工需指定主要部門 | DR-009、DR-011 |
| ENT-003 | LeaveRequest（請假申請） | 員工發起的假期申請單，有完整的狀態生命週期 | DR-001 ~ DR-005 |
| ENT-004 | ReviewRecord（審核紀錄） | 記錄主管對請假申請的審核動作與結果 | DR-006、DR-007 |
| ENT-005 | LeaveBalance（假期餘額） | 記錄員工各假別的剩餘天數 | DR-008 |
| ENT-006 | ManagerDelegate（代理主管設定） | 記錄主管指定的代理人與代理期間 | DR-010 |

## Relationships

| Relationship ID | 關係 | 描述 | 來源 |
| --- | --- | --- | --- |
| REL-001 | Employee → Department | 員工屬於一個主要部門（跨部門員工仍需指定主要部門） | BR-009、DR-009 |
| REL-002 | Employee → LeaveRequest | 員工發起請假申請（一對多） | FR-001 |
| REL-003 | Employee → LeaveBalance | 員工擁有各假別的假期餘額記錄（一對多，每假別一筆） | DR-008 |
| REL-004 | Manager → LeaveRequest | 主管（或代理主管）審核管轄員工的請假申請（產生 ReviewRecord） | FR-003、PR-004 |
| REL-005 | LeaveRequest → ReviewRecord | 一筆申請對應一筆審核紀錄 | FR-003、DR-007 |
| REL-006 | Manager → ManagerDelegate | 主管可設定代理主管（一對一代理設定，包含期間） | FR-008、DR-010 |
| REL-007 | Department → Manager | 每個部門有一位負責審核的主管 | BR-009 |

## 實體屬性詳細說明

### ENT-001：Employee（員工）

| 屬性 | 說明 | 來源 |
| --- | --- | --- |
| ID | 員工唯一識別碼 | 系統基本需求 |
| 姓名 | 員工姓名 | DR-011 |
| 主要部門 | 跨部門員工的主審部門 | DR-009、BR-009 |
| 角色 | Employee / Manager / HR（可複合） | ROLE-001 ~ ROLE-003 |
| 年資 | 用於計算特休天數 | BR-004 |

### ENT-003：LeaveRequest（請假申請）

| 屬性 | 說明 | 來源 |
| --- | --- | --- |
| ID | 申請唯一識別碼 | — |
| 申請員工 | 發起申請的員工 | FR-001 |
| 假別 | 特休 / 病假 / 事假 / 喪假 | DR-003 |
| 開始日期 | 請假起始日 | DR-001 |
| 結束日期 | 請假結束日 | DR-002 |
| 天數 | 以 0.5 天為最小單位計算 | BR-008 |
| 原因 | 申請原因（必填） | DR-004 |
| 附件 | 病假超過 3 天時必填 | DR-005、BR-007 |
| 狀態 | 待審核 / 已核准 / 已拒絕 / 已取消 | WR-001 ~ WR-005 |
| 申請時間 | 申請送出的時間戳記 | DR-011 |

### ENT-004：ReviewRecord（審核紀錄）

| 屬性 | 說明 | 來源 |
| --- | --- | --- |
| ID | 審核紀錄唯一識別碼 | — |
| 關聯申請 | 對應的 LeaveRequest | FR-003 |
| 審核人 | 執行審核的主管（含代理主管） | DR-007 |
| 審核結果 | 同意 / 拒絕 | FR-003 |
| 拒絕原因 | 拒絕時必填 | DR-006、BR-005 |
| 審核時間 | 審核動作的時間戳記 | DR-007 |

### ENT-005：LeaveBalance（假期餘額）

| 屬性 | 說明 | 來源 |
| --- | --- | --- |
| 員工 | 對應員工 | DR-008 |
| 假別 | 特休 / 病假 / 事假 / 喪假 | DR-008 |
| 剩餘天數 | 不得為負數，最小單位 0.5 天 | DR-008、BR-008 |
| 年度 | 假期餘額所屬年度 | BR-001 ~ BR-004 |

### ENT-006：ManagerDelegate（代理主管設定）

| 屬性 | 說明 | 來源 |
| --- | --- | --- |
| 被代理主管 | 設定代理的主管 | DR-010 |
| 代理人 | 被指定執行代理的員工 | DR-010 |
| 代理起始日 | 代理期間開始日期 | DR-010 |
| 代理結束日 | 代理期間結束日期，不得早於起始日 | DR-010 |
