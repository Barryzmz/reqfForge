# Acceptance Criteria

本文件彙整各需求的驗收條件，詳細 Given/When/Then 以 `specs/02-requirements/functional-requirements.md` 為主，此處為摘要索引。

| AC ID | 關聯需求 | 情境摘要 | 狀態 |
| --- | --- | --- | --- |
| AC-001 | FR-001 | 員工填妥所有必填欄位後送出，系統建立「待審核」申請 | Confirmed |
| AC-002 | FR-001 | 員工缺漏必填欄位，系統拒絕送出並提示缺漏項目 | Confirmed |
| AC-003 | FR-001 | 系統支援以 0.5 天為最小申請單位 | Confirmed |
| AC-004 | FR-002 | 員工申請病假超過 3 天且未上傳診斷書，系統拒絕送出 | Confirmed |
| AC-005 | FR-002 | 員工申請病假 3 天以內，系統允許不附診斷書送出 | Confirmed |
| AC-006 | FR-002 | 員工申請病假超過 3 天並上傳診斷書，系統允許送出 | Confirmed |
| AC-007 | FR-003 | 主管選擇同意，申請狀態更新為「已核准」，審核時間與審核人已記錄 | Confirmed |
| AC-008 | FR-003 | 主管選擇拒絕並填寫原因，申請狀態更新為「已拒絕」，拒絕原因已記錄 | Confirmed |
| AC-009 | FR-003 | 主管選擇拒絕但未填寫原因，系統拒絕送出 | Confirmed |
| AC-010 | FR-004 | 申請送出後超過 3 天仍待審核，系統向主管發送系統內通知與 Email | Confirmed |
| AC-011 | FR-005 | HR 登入後查看假期紀錄，系統顯示所有員工的請假申請紀錄 | Confirmed |
| AC-012 | FR-006 | HR 匯出報表，系統產出包含指定欄位的 Excel 檔案 | Confirmed |
| AC-013 | FR-007 | 員工取消「待審核」申請，申請狀態更新為「已取消」，從主管待審核清單移除 | Confirmed |
| AC-014 | FR-007 | 員工嘗試取消「已核准」或「已拒絕」申請，系統拒絕操作 | Confirmed |
| AC-015 | FR-008 | 主管設定代理人及代理期間，代理期間內代理人可審核被代理主管管轄員工的申請 | Confirmed |
| AC-016 | FR-008 | 代理期間結束後，代理人嘗試審核相關申請，系統拒絕操作 | Confirmed |
| AC-017 | BR-005 | 主管拒絕申請時若未填寫拒絕原因，系統不允許完成審核動作 | Confirmed |
| AC-018 | BR-006 | 申請超過 3 天未審核觸發催審，申請狀態不因催審而改變 | Confirmed |
| AC-019 | BR-007 | 病假申請天數 ≤ 3 天，診斷書非必填；> 3 天，診斷書必填 | Confirmed |
| AC-020 | BR-009 | 跨部門員工的申請僅由其主要部門主管可審核 | Confirmed |
| AC-021 | PR-002 | 員工登入後只能看到自己的請假紀錄，無法存取其他員工的紀錄 | Confirmed |
| AC-022 | PR-005 | 代理主管在代理期間外嘗試審核，系統拒絕操作 | Confirmed |
